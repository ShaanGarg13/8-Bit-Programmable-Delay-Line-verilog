# 8-Bit-Programmable-Delay-Line-verilog
Implementation of an 8-Bit Programmable Delay Line in Verilog HDL inspired by the Data Delay Devices 3D3418 programmable delay line, supporting serial and parallel delay programming, configurable delay control, shift register interfaces, RTL design, simulation, waveform analysis, and logic synthesis using Quartus II and ModelSim-Altera.
